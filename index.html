Tweakpane

Compact pane library for fine-tuning parameters and monitoring value changes.

Concept

Clean and simple design

Easy to understand API, modest appearance that brings out a main content.

Dependency-free

Stand-alone, doesn't require any other library like jQuery, React, Vue.js, etc.

Extensible

Highly customizable with theming and the plugin system.
